# Top Multiplayer PC Games

## Overview

This project is part of an NLP Lab assignment. It focuses on building a custom dataset by collecting and processing data from Wikipedia. The dataset contains textual information about multiplayer PC games.

## Process

* Collected game titles using Wikipedia search
* Filtered only actual video game pages
* Saved each game in separate `.txt` files
* Merged all files into a single CSV dataset

## Files

* `scraper.py` → Data collection script
* `merge.py` → Data merging script
* `Top_Multiplayer_PC_Games_Dataset.csv` → Final dataset

## Output

A clean dataset ready for NLP tasks such as text analysis and preprocessing.
